# Data Visualization

## Description
This repository contains scripts used to visualize bioinformatics data, including a Shiny app built with R and basic `ggplot2` visualizations. The goal is to improve my data visualization skills for genomic datasets and epidemiological data.

## Learning Goals
- Develop my skills in R and `ggplot2` for effective data visualization.
- Learn how to build interactive visualizations with Shiny.

## Files
- `Shiny_App.R`: An interactive Shiny app for visualizing sequence data.
- `ggplot.sh`: A script that uses `ggplot2` to create basic visualizations.

## Usage
To run the Shiny app:
```R
shiny::runApp("Shiny_App.R")
